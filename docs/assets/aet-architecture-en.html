Agent Engineering Toolkit (AET) — Architecture Overview

Agents do the work. AET records evidence, enforces boundaries, and keeps long-term improvement behind independent evaluation and human approval.

Use the smallest surface

Start with the fact you need. Use Review for scope, Trace for proof execution, and ordinary project tests/CI for routine low-risk work.

Do not collapse states

Command success, artifact presence, current freshness, and scope compliance are different facts. Missing evidence remains UNKNOWN.

Remove cost without removing evidence

Preflight first; resume/reuse exactly; keep Held-out closed after an earlier failure; size evaluation by risk and claim, never a fixed run count.

Keep authority asymmetric

Agents work and candidates enter evaluation, but candidates cannot rewrite judges, inspect held-out early, or approve themselves. Humans adopt after Stage.
