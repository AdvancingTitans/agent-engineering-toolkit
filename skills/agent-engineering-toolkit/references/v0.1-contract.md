# v0.1 contract

`aet audit` discovers `AGENTS.md`, `CLAUDE.md`, `CODEX.md`,
`copilot-instructions.md`, `.cursorrules`, and `SKILL.md` files. It reports
broken local Markdown and explicit command targets, instruction bloat and
duplication, and Skill frontmatter/name/verification problems. Bare filenames
and dynamic output names are not treated as local references.

Use Markdown for a human handoff, JSON for an Agent evidence bundle, and SARIF
for code-scanning integration. Default mode fails on `FAIL`; `--strict` also
fails on `WARN`.
